# Array ou vetores

Uma estrutura básica que muitas vezes já é nativa das linguagens, em Python como Lista, em C como Vetor, 
onde através de um índice (0 a n).

Há também algumas bibliotecas que podem trazer outros tipos de arrays e métodos como o NumPy.


## Achar o menor valor de um array

Um dos trabalhos mas frenquentes é achar um determinado valor num array. Um deles é achar o menor
valor de um array.

A forma mais simples é atravessar o array e comparar o valor com cada outro valor do array. Se
for um valor menor então substituir o anterior.

## Algoritmo - Achar o menor valor

1. Determinar o primeiro valor como o menor;
2. Atravessar o array;
3. Se o valor for menor que o atual menor valor, substituir.
