# Introdução - Data Structures and Algorithms ou Estrutura de Dados e Algoritmos

Quando você precisar criar código e programas maiores e mais complexos, 
há uma nítida necessidade de torna seu projeto minimamente organizado, 
minimamente planejado. Para tanto, é preciso estudar algoritmos e estrutura de dados.

**Algoritmos**, como você já pode estar cansado de saber, é um conjunto de instruções bem
definidos, finitos e objetivo. Eles nós conceitua o processo lógico de chegar numa
solução ou, manipular dados.

Outro problema que surge é como armazenar e manipular, grandes quantidades de dados.
Tendo em mente que, vivemos num período crescente na obtenção, extração e manipulação
de dados, **Estrutura de Dados** é o campo que estuda como armazenar e manipular dados 
eficientemente.
